To run this application properly, please follow the following steps:

1. Navigate to  /src->/main->/JDBC
2. Open the file "Connect"
3. In there, enter your personal database password and username into the Strings AUTH_STRING and USER.

4. To run this application, navigate to /src->/main->/eHealth_GUI
5. In this folder, run the file "Healthcare_Login". This will open the starting window of our application.
